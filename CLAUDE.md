# CLAUDE.md

## 作業ルール（絶対厳守）

- 作業開始前に必ず `REQUIREMENTS_v2.md`（要件定義）と `PROGRESS.md`（進捗管理）の内容を確認してから作業を進めること。
- 作業を行うたびに `PROGRESS.md` を更新すること（完了事項・次にやること・保留中の意思決定事項を反映する）。
- `REQUIREMENTS_v2.md`（要件定義ファイル）の内容を更新する際は、事前にユーザーの許可を取ってから行うこと。無断で要件定義を変更しない。

## 関連ファイル

- `System_Detailed_Spec_JP.md` : 初期仕様書（旧版、参考程度）
- `REQUIREMENTS_v2.md` : 最新の要件定義ドラフト（インフラ構成・予測ロジック設計を含む）
- `FEATURES.md` : 機能一覧（技術スタックを含まない、ユーザー視点の機能リスト）
- `PROGRESS.md` : 進捗管理ファイル（作業のたびに更新する）
